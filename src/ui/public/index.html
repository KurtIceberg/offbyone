推荐第一步 · Plan Mode / 构建计划

Plan Mode 会整理项目名称、目标用户、语言策略、1–3 页页面列表、区块大纲、视觉方向、图片策略、CTA 与质量清单。

Recommended / 质量优先
Plan Mode 默认开启：首次 Build 会先生成结构化 site brief；确认后再生成 1–3 页网站。需要跳过时可关闭 Plan Mode。
运行策略与输出设置 / Run options
可选真实大模型参数（不会接收 API Key）

API Key 只从服务端环境变量读取，不会从此表单接收，也不会发送到浏览器。

Plan Mode 开启时会先停在 site brief 确认；关闭 Plan Mode 才会直接生成。
